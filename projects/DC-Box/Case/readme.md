# Housing

#### Things used for the housing:
- 4mm acrylic glass 600x400mm
- laser cutter
- screws and nuts for Raspberry PI
- hot glue gun

2) The nuts are fastened in the housing.
3) The Raspberry PI is screwed in.
4) The PIcamera is assembled.
